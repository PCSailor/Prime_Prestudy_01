Bonus Challenge:
TO-DOMore functions!

Ok, console logs are great, but lets actually make functions that do things. Create three functions, each taking in a single number. Then do something to that number and return a new number. Then call each of those functions in one giant chain. Assuming that you set this up correctly, you should get a numerical return that is a combination of all the functions running, taking in a function in the place of a number. EVEN IF YOU GET THIS WORKING AND ARE UNSURE OF 'WHY' IT WORKS, thats OK. This is just showing you that you can use functions in place of many things. We will dive deep into these concepts in class!

Example:
add(multiply(subtract(2)));
Function Name What It Does
add Add 2 to the number
multiply Multiply the number by 3
subtract Subtract 4 from the number

Once you are complete, save it to a file named "scripts.js" and check it into a repo titled "prime_prestudy_bonus_week_03"
